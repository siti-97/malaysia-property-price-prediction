# malaysia-property-price-prediction

-------------------
Dataset Description
--------------------
(i)	Dataset Source

The dataset comes from Kaggle (octopus69/malaysia-residential-real-estate) and was originally scraped from EdgeProp.my, one of Malaysia's leading property websites. It includes residential listings across all thirteen states and three federal territories in Malaysia.

(ii)	Organizational Problem

The Malaysian real estate market suffers market suffers from information asymmetry or in other words lack of clear data, and also manual valuation inconsistencies. For instance, buyers and investors lack the inability to rapidly and accurately estimate fair market values for residential properties, which leads to mispricing risks for property portfolios and poor investment decisions. Hence, this project aims to address the problem by developing a predictive regression model that quantifies fair market value based on structural and geospatial features.

(iii)	Intended Stakeholders
-----------------------------------------------
Stakeholders	          - Benefit / Use Case
------------------------------------------------
Individual Home Buyers	- Identify overpriced listings and negotiate confidently using data-backed baseline prices.
Real Estate Investors	  - Spot undervalued properties and identify high-ROI opportunities across different states.
Bank Valuers	          - Obtain automated, objective baseline valuations for collateral and mortgage risk assessments.
Property Developers	    - Inform future site selection and competitive pricing strategies based on regional demand.

(iv)	Number of Records
7,353 records of property listings.

(v)	Number of Variables
48 variables, including location, structural details, and proximity metrics.

(vi)	Target Variable
Price in Malaysian Ringgit that refers to the listed market price of the property and it is a continuous variable.

(vii)	Type of Analysis
Exploratory Data Analysis (EDA) is performed to understand feature distributions, relationships, and quality issues, followed by supervised machine learning with regression algorithms to predict property prices (continuous numerical target). Then, model performance will be evaluated using standard error metric such as RMSE and R².
